Get Started With EasyElegance

Welcome to EasyElegance, a comprehensive library built using HTML, SASS, JS, and Gulp as the task runner. This library is designed to streamline your web development process by offering a collection of reusable components, flexible grid systems, and versatile utilities. Whether you're creating a simple webpage or a complex web application, EasyElegance provides the tools you need to craft beautiful and responsive designs with ease.

Layout

In the layout section of EasyElegance, you'll find a robust grid system tailored to accommodate various screen sizes and resolutions. With classes for both parent containers and child elements, along with predefined breakpoints for media queries, designing responsive layouts has never been simpler. Whether you're building a single-column blog or a multi-column e-commerce site, our grid system empowers you to create visually stunning and user-friendly interfaces effortlessly.

Breakpoints

Explore the power of media queries with our carefully curated breakpoints. Seamlessly transition between different screen sizes and resolutions by leveraging our predefined breakpoints, allowing you to fine-tune your layouts for desktops, tablets, and mobile devices with ease. With clear and intuitive naming conventions, customizing your designs for specific breakpoints has never been simpler.

Margin And Padding

Fine-tune the spacing and layout of your elements with our comprehensive margin and padding utilities. Whether you need to add breathing room between content sections or adjust the alignment of individual elements, our utilities provide precise control over spacing with intuitive classes. With support for responsive design and shorthand notation, achieving consistent and visually appealing layouts has never been easier.

How to use Margin and Padding

The classes are named using the format {property}{sides}-{size}.

Where property is one of:

  • m - for classes that set margin
  • p - for classes that set padding

Where sides is one of:

  • t - for classes that set margin-top or padding-top
  • b - for classes that set margin-bottom or padding-bottom
  • l - for classes that set margin-left or padding-left
  • r - for classes that set margin-right or padding-right
  • x - for classes that set both *-left and *-right
  • y - for classes that set mboth *-top and *-bottom
  • blank - for classes that set a margin or padding on all 4 sides of the element

Where size is one of:

  • 0 - for classes that eliminate the margin or padding by setting it to 0
  • 1 - (by default) for classes that set the margin or padding to $spacer .75rem
  • 2 - (by default) for classes that set the margin or padding to $spacer 1.5rem
  • 3 - (by default) for classes that set the margin or padding to $spacer 3rem
  • 4 - (by default) for classes that set the margin or padding to $spacer 4.5rem
  • 5 - (by default) for classes that set the margin or padding to $spacer 6rem
  • auto - for classes that set the margin to auto
